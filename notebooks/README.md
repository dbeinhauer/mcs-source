# About
This directory contains examples of several result and dataset analysis workflows that has been done in the previous research. It may serve as a tutorial on how to visualize the model predictions or dataset analyses.

# Structure of Directory
- `conference_plots/` - Plotting workflow done for a visualizations for a conference posters.
- `interpret_model.ipynb`, `visualize_features.ipynb` - Several analyses done for evaluation results.
- `wandb/` - W&B results analyses
