# About
This directory contains scripts for running the `evaluation_processor.py` functionality on
the Wintermute cluster. 

# Directory Content
The directory contains the following scripts:
- `evaluation_analysis.sh` - Script to run analysis of the raw evaluation results (direct results from the model).
- `full_dataset_analysis.sh` - Script to run analysis of the full dataset on different time bin sizes.
- `subset_dataset_analysis.sh` - Script to run analysis of the different dataset subset variants.
